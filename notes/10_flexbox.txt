
    Flexbox
            เครื่องมือใน css ที่ช่วยให้การจัดการ element ต่าง ๆ ในหน้าเว็บมีความง่ายขึ้นและยืดหยุ่นมากขึ้น โดนทั่วไปการจัดตำแหน่ง element 
        ต่าง ๆ ต้องใช้ layout mode คือ block, inline, position และอื่น ๆ 
        โดยในปัจจุบันเว็บมีความซับซ้อนมากขึ้น ทำให้มีการพัฒนา Flexbox ขึ้นมาเพื่อจัดการ element ให้มีความยืดหยุ่นสูงโดยมีคุณสมบัติคือ
            - จัดเรียงตำแหน่งของ element ได้ง่ายขึ้น จากบนลงล่าง ซ้ายไปขวา และอื่น ๆ
            - กำหนดขนาดให้พอดีกับพื้นที่ว่างโดนอัตโนมัติ (Sizing)

        * องค์ประกอบของ Flexbox คือ container และ item

        * Flex Direction
        - เป็นการกำหนดทิศทางของ element
        - class:
            - row               จัดการ items ในแนวนอนให้เรียงตามแกนหลัก
            - row-reverse       จัดการ items ในแนวนอนให้เรียงตรงกันข้ามกับแกนหลัก
            - column            จัดการ items ในแนวตั้งทิศทางเดียวกับแกนหลัก
            - column-reverse    จัดการ items ในแนวตั้งให้เรียงตรงกันข้ามกับแกนหลัก
        - example:
            - flex-row-reverse
            - flex-col
            - flex-col-reverse

        * Wrap FlexItem
        - กรณีที่ขนาด items ใหญ่กว่าพื้นที่ container
        - class:
            - nowrap        จัดวาง items ที่เกินพื้นที่ container ไปด้านขวามือ
            - wrap          จัดวาง item ที่เกินพื้นที่ container เรียงจากบนลงล่าง
            - wrap-reverse  จัดวาง item ที่เกินพื้นที่ container เรียงจากล่างขี่นบน  

        * Grow & Shrink
        - class:
            - flex-1    ทำให้ item ที่อยู่ในแถวเดียวกันมีขนาดเท่ากัน
            - Shrink    ให้ item หดตัวจำนวนเท่าใดเมื่อเทียบกับ item อื่น ๆ (ค่าเริ่มต้นเป็น 1)
            - grow      ให้ item ขยายจำนวนเท่าใดเมื่อเทียบกับ item อื่น ๆ (ค่าเริ่มต้นเป็น 0)
